Instrucciones de configuración Microsoft 365

1. Ve a portal.azure.com → Azure Active Directory → App Registrations → New registration

2. Nombre: CapaSync. Redirect URI: https://tu-servidor.com/auth/callback

3. API Permissions → Add: Sites.ReadWrite.All, Files.ReadWrite.All, User.Read

4. Certificates & Secrets → New client secret → copia el valor

5. En SharePoint, crea una carpeta y sube Capacitaciones.xlsx con las columnas indicadas arriba

6. Ingresa Tenant ID y Client ID en los campos de arriba y guarda. El sistema usará MSAL.js para autenticación sin contraseñas.
